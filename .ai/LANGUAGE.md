# Project Vocabulary

Canonical product and codebase terms for AI agents. Use the canonical term;
avoid the listed synonyms so naming stays consistent across code and docs.

## Terms

| Term | Meaning | Avoid |
| --- | --- | --- |
| Materials | The raw source a user provides (text/notes) that the app transforms into a post. | input, content, source text |
| Post | The generated, platform-ready output (copy/description + supporting text) the user copies to publish. | article, output, result |
| Platform | The target social channel (LinkedIn, TikTok, YouTube, …) whose audience determines tone and format. | app, channel, network |
| Tone profile | The platform-specific style the post is tailored to (e.g. professional vs. entertaining). | persona, voice setting |
| Generation | The server-side Claude step that produces a post from materials. | processing, AI call |
| Repurposing | Adapting one set of materials into platform-tailored posts. | conversion, reformatting |
